and purcahse product.

1 - left menu 'SHOP COLLECTION -> SUB MENUS'
2 - left menus FROM 'THE EMBRACE COLLECTOIN' TO 'CONTACT US'
3 - SHOPPING CART
4 - CHECKOUT MENU
5 - PAYMENT METHOD
6 - it has 5 menus on the top right corner ---
    "SHOP, LOGIN, SIGNUP, CART, CURRENY DROPDOWN MENU"
